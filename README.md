# Adaptive Approximate Multiplier using Edge Detection for Image Enhancement

## Overview

This project implements an Adaptive Approximate Multiplier for image enhancement applications using Verilog HDL and MATLAB.

Approximate computing reduces hardware complexity, power consumption, and delay while maintaining acceptable image quality. The proposed approach uses edge detection to identify important image regions and selectively applies exact or approximate multiplication.
